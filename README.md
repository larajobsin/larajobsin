# Installation

Installation steps
```
	git clone https://github.com/rajentrivedi/lara-jobs.git
```

```
	composer install
```
```
	php  artisan  filament:install  --scaffold  --forms
```
```
	npm install
```
```
	npm run dev
```
```
	php artisan key:generate
```
```
	php artisan migrate
```
```
	php artisan db:seed
```
```
	php artisan  storage:link
```
